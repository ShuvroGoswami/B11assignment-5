Question-2: What is the difference between innerHTML, innerText, and textContent ?
"innerHTML" retrieves the complete HTML content within an element, including tags, while "innerText" only retrieves the visible text content of an element (ignoring HTML tags), and "textContent" retrieves all text content within an element, including hidden text, but without interpreting any HTML tags.
Question-3: What is event delegation in the DOM?
DOM (Document Object Model), "event delegation" is a technique where you attach a single event listener to a parent element, allowing it to handle events triggered on any of its child elements.
Question-4: What is event bubbling in the DOM?
the behavior where when an event occurs on an element, it triggers not only on that element but also propagates upwards through its parent elements in the DOM hierarchy
